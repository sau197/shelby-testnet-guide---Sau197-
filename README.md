# Testnet Guide

## Step 1: Install Aptos Wallet

Install Petra Wallet

https://chromewebstore.google.com/detail/petra-aptos-wallet/ejjladinnckdgjemekebdpeokbikhfci

After installing:
- Create wallet
- Save recovery phrase

---

## Step 2: Get Faucet Token

Claim test tokens:

APT Testnet  
ShelbyUSD

Guide:
https://docs.shelby.xyz/tools/wallets/petra-setup

---

## Step 3: Upload File

Go to Shelby Explorer:

https://explorer.shelby.xyz/testnet

Steps:

1 Connect Petra Wallet  
2 Click wallet address (top right)  
3 Upload file  
4 Confirm transaction

---

## Notes

- Switch wallet network to Shelby Testnet
- Upload multiple files
- Generate more transactions
