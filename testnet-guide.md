# Shelby Testnet Detailed Guide

This document explains how to interact with Shelby Testnet.

---

## Install Wallet

Install Petra Wallet.

Create wallet and switch network to Shelby Testnet.

---

## Get Test Tokens

Use faucet to get:

APT testnet
ShelbyUSD

---

## Upload Files

Go to explorer.

Connect wallet.

Upload images or files.

Confirm transaction.

Repeat multiple times.
